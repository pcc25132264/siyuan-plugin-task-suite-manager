# 任务管理中心（Task Suite Manager）

这是一个面向思源的综合任务管理插件，提供统一的数据模型与多视图协同管理能力。

## 核心功能

- 清单管理：任务增删改查、优先级、状态、标签、子任务、重复规则（一次/每日/每周/每月）
- 时间轴：历史变更与未来计划统一展示，支持时间筛选与快速定位
- 看板视图：拖拽式任务流转，自定义列映射状态，实时同步
- 日历视图：月视图/周视图/日视图，支持在日期上快捷创建任务
- 甘特图：任务进度可视化、依赖关系、关键路径识别、资源分配展示

## 数据互通说明

插件使用统一任务状态存储，任意视图中的变更会即时写回并同步到其他视图：

- 看板拖拽会更新任务状态，并在清单/时间轴/甘特图中立即反映
- 清单中编辑时间与依赖关系，会同步影响日历与甘特图表现
- 子任务勾选可自动刷新主任务进度与状态

## 使用方式

1. 在思源顶部栏点击“任务管理中心”图标，或使用命令“打开任务管理中心”
2. 在“清单”页创建任务并配置时间、优先级、依赖和资源
3. 切换到“看板/日历/甘特图/时间轴”按不同视角管理同一批任务
4. 点击“运行自测”可进行基础结构与重复任务逻辑的快速检查

## 插件信息

- 插件名：`siyuan-plugin-task-suite-manager`
- 显示名：任务管理中心
